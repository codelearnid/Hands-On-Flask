Halo! Berikut adalah referensi perintah untuk menggunakan Command Line pada
komputer Mac OS atau Linux. Perlu diingat, ini hanya perintah dasar yang
seharusnya Anda ketahui, ada banyak perintah lainnya selain ini.

----------------------------
--- Mengubah Direktori -----
----------------------------

Gunakan cd untuk mengubah direktori. Misalnya, jika Anda berada di folder utama untuk
semua catatan dan Anda menyimpannya ke folder unduhan Anda dan Anda ingin
mengubah ke direktori ini, maka akan terlihat seperti ini:

Joses-Macbook-Pro:~ Jose:  cd Desktop

akan menghasilkan:

My-MacBook-Pro:Desktop Jose$

Ingat bahwa Anda biasanya dapat menggunakan tombol Tab pada keyboard untuk
mengisi otomatis jalur file!

Perlu kembali ke folder atas? Itu mudah, hanya cd diikuti oleh dua titik/periode:

My-MacBook-Pro:Desktop Jose$ cd ..

akan menghasilkan:

My-Macbook-Pro:~ Jose:

-----------------------------------------
--- Menampilkan Direktori Saat Ini ---
-----------------------------------------

Untuk sebagian besar terminal Anda seharusnya dapat melihat direktori Anda
langsung di sebelah kiri kursor Anda, tetapi jika Anda memiliki keraguan, cukup gunakan
pwd sendiri untuk mencetak direktori kerja Anda:


Joses-Macbook-Pro:~ Jose: pwd

akan menghasilkan:

/Users/Jose

--------------------------------------------------------
--- Menampilkan Semua Folder di Direktori Saat Ini ---
--------------------------------------------------------

Anda dapat menggunakan ls untuk menampilkan semua file dan folder di direktori saat ini:

My-MacBook-Pro:Downloads Jose$ ls

ini akan menghasilkan output dari semua file dan folder di dalam folder Downloads.

----------------------------
--- Membersihkan Layar ----
----------------------------

Anda dapat menggunakan clear untuk membersihkan layar prompt perintah. Anda dapat selalu menekan tombol panah atas untuk mendapatkan kembali perintah yang lebih lama (walaupun Anda tidak dapat menghapus perintah)
